# 哲学家陪聊 · 《被讨厌的勇气》

四个哲学家陪你聊天：阿德勒、庄子、苏格拉底、尼采。

## 功能

- 用户提问，四个哲学家依次回答
- 先安慰，后教学
- 支持切换“只听TA”模式

## 在线体验

https://philosopher-chat.vercel.app（部署后生效）
